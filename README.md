Nome do Projeto: Gerenciador De Tarefas

Objetivo: Garantir uma melhor organização de objetivos no dia a dia do usuário.

Status do Projeto: Em andamento

Tecnologias aplicadas: 
= Java
= MySQLite
= GitHub
= Figma

Funcionalidades do Projeto: O sistema terá uma tela onde o usuário pode adicionar tarefas à lista, definindo seu grau de prioridade. Na lista, o usuário pode atualizar a tarefa, excluir ou marcá-la como concluída.
